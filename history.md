# 版本历史

## 1.0.14 (2026-03-20)
- 增强 DWSIM Agent 自动化能力：优化意图识别和参数提取
- 新增 DWSIM 特定意图分类：支持创建流程图、运行仿真、灵敏度分析、优化、添加设备等
- 增强 task_understanding.py：改进 DWSIM 任务识别和执行计划生成
- 完善 dwsim_com_adapter.py：添加安全连接机制和错误处理
- 新增 DWSIM 相关提示词模板：意图分类、参数提取、设备映射、任务规划

## 1.0.13 (2026-03-20)
- 修复图片分析功能：解决导出内容为空的问题
- 修复图片预览接口：支持通过URL参数传递token，解决401未授权错误
- 优化图像解析器资源等级：将资源等级改为low，确保所有用户都能使用
- 优化OCR提取逻辑：确保提取的文本正确传递到前端

## 1.0.12 (2026-03-20)
- 新增 PDF → Word 翻译功能：使用 pdf2docx 转换 PDF 为 Word，保留原文格式和排版
- 新增文件下载 API：支持下载翻译后的 Word 文档
- 优化短文本翻译速度：使用 HY-MT1.5-1.8B 模型，翻译速度提升 60 倍（<1秒）
- 优化智能对话速度：禁用 Qwen3.5 thinking 模式，响应速度提升 10 倍
- 新增前端 PDF → Word 翻译卡片：支持源语言选择、目标语言选择、进度显示、文件下载

## 1.0.4 (2026-03-09)
- 修复参数映射页面：添加缺失的 parameter-mappings API 路由
- 新增参数映射管理功能：支持创建、更新、删除、切换启用状态
- 新增批量导入导出功能：支持批量创建参数映射

## 1.0.3 (2026-03-08)
- 修复文档管理页面：添加缺失的 report-templates 预览 API 路由
- 修复文档管理页面：在 ReportGenerator 类中添加 get_template_preview 方法
- 新增默认报告模板：创建 default.html 报告模板文件

## 1.0.2 (2026-03-08)

## 1.0.1 (2026-03-08)
