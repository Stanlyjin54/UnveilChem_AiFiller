# Aifiller 用户使用说明

## 1. 产品简介

Aifiller 是一款专业的化学文档参数提取工具，能够自动识别和提取化工文档中的关键参数，提高工作效率和准确性。

### 1.1 主要功能
- 支持多种文档格式（PDF、图片、Office文档等）
- 自动提取化工参数和表格
- 高精度OCR识别
- 复杂PDF解析
- 批量处理功能

## 3. 快速开始

### 3.1 注册与登录
1. 访问 Aifiller 网站
2. 点击 "注册" 按钮，填写注册信息
3. 使用注册的账号密码登录

### 3.2 上传文档
1. 登录后，点击 "上传文档" 按钮
2. 选择要上传的文档文件
3. 等待上传完成

### 3.3 解析文档
1. 上传完成后，系统会自动开始解析
2. 解析完成后，查看解析结果
3. 可以对解析结果进行编辑和调整

### 3.4 导出结果
1. 解析完成后，点击 "导出" 按钮
2. 选择导出格式（CSV、Excel、JSON等）
3. 下载导出文件

## 4. 详细功能说明

### 4.1 文档上传
- 支持单文件上传和批量上传
- 支持拖拽上传
- 显示上传进度

### 4.2 解析设置
- 选择解析模式（快速解析/高精度解析）
- 设置解析区域
- 选择要提取的参数类型

### 4.3 结果查看
- 查看提取的参数列表
- 查看提取的表格
- 查看原始文档和解析结果对比

### 4.4 结果编辑
- 修改提取的参数值
- 添加或删除参数
- 调整表格结构

### 4.5 历史记录
- 查看历史解析记录
- 重新解析历史文档
- 删除历史记录

## 5. 常见问题

### 5.1 文档解析失败怎么办？
- 检查文档格式是否支持
- 检查文档是否损坏
- 尝试使用高精度解析模式
- 联系技术支持

### 5.2 解析结果不准确怎么办？
- 调整解析区域
- 手动编辑解析结果
- 尝试使用不同的解析器

### 5.3 如何提高解析准确率？
- 确保文档清晰可读
- 避免使用手写文档
- 确保文档中的表格结构清晰
- 使用专业版或企业版，享受更高精度的解析

### 5.4 配额用完了怎么办？
- 等待下月自动重置
- 升级到更高版本
- 购买额外配额
