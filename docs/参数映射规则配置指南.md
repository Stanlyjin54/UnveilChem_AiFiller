# 软件参数映射规则配置指南

## 概述

UnveilChem智能参数录入助手提供了灵活的软件适配器架构，允许为不同软件创建专用的参数映射规则。本文档详细说明如何配置和管理这些映射规则。

## 参数映射架构

### 1. 核心组件

```python
from backend.app.services.automation.parameter_mapper import (
    ParameterMapper, ParameterType, SoftwareType, ParameterMapping
)
```

### 2. 支持的软件类型

| 软件类型 | 标识符 | 类别 | 主要用途 |
|---------|--------|------|----------|
| Aspen Plus | `aspen_plus` | 化工流程模拟 | 工艺参数、流程优化 |
| DWSIM | `dwsim` | 化工流程模拟 | 流程仿真、物性计算 |
| PRO/II | `pro_ii` | 化工流程模拟 | 工艺设计、成本分析 |

```python
        self.add_mapping(ParameterMapping(
            software_type=SoftwareType.ASPEN_PLUS,
```

## 高级映射配置

### 1. 条件映射

```python
class ConditionalParameterMapper(ParameterMapper):
    """条件参数映射器"""
    
    def add_conditional_mapping(self, param_name: str, condition: callable, mapping: ParameterMapping):
        """添加条件映射"""
        self.conditional_mappings[param_name] = (condition, mapping)
    
    def get_mapping_with_conditions(self, param_name: str, context: dict) -> Optional[ParameterMapping]:
        """获取条件映射"""
        if param_name in self.conditional_mappings:
            condition, mapping = self.conditional_mappings[param_name]
            if condition(context):
                return mapping
        return None
```

### 2. 批量映射

```python
class BatchParameterMapper(ParameterMapper):
    """批量参数映射器"""
    
    def add_batch_mappings(self, software_type: SoftwareType, param_configs: List[dict]):
        """批量添加映射"""
        for config in param_configs:
            mapping = ParameterMapping(
                software_type=software_type,
                parameter_type=ParameterType(config['parameter_type']),
                software_field=config['software_field'],
                display_name=config['display_name'],
                unit_conversion=config.get('unit_conversion', {}),
                validation_rules=config.get('validation_rules', {})
            )
            self.add_mapping(mapping)
```

## 参数验证规则

### 1. 数值验证

```python
def create_numeric_validation(min_val: float, max_val: float, required: bool = True):
    """创建数值验证规则"""
    return {
        "type": "number",
        "required": required,
        "min": min_val,
        "max": max_val,
        "precision": 6  # 精度限制
    }
```

### 2. 字符串验证

```python
def create_string_validation(pattern: str = None, max_length: int = None):
    """创建字符串验证规则"""
    validation = {
        "type": "string",
        "pattern": pattern if pattern else None,
        "maxLength": max_length
    }
    return validation
```

### 3. 枚举验证

```python
def create_enum_validation(allowed_values: List[str]):
    """创建枚举验证规则"""
    return {
        "type": "enum",
        "allowed": allowed_values
    }
```

## 使用示例

### 1. 创建自定义映射器

```python
from backend.app.services.automation.parameter_mapper import ParameterMapper, ParameterMappingRegistry

# 创建自定义映射器
class CustomSoftwareMapper(ParameterMapper):
    def __init__(self):
        super().__init__()
        # 添加自定义映射规则
        self.add_mapping(ParameterMapping(
            software_type=SoftwareType.ASPEN_PLUS,
            parameter_type=ParameterType.CUSTOM,
            software_field="CUSTOM_FIELD",
            display_name="自定义参数",
            unit_conversion={},
            validation_rules={"type": "string", "maxLength": 50}
        ))

# 注册映射器
registry = ParameterMappingRegistry()
registry.register_mapper("custom_software", CustomSoftwareMapper())
```

## 配置最佳实践

### 1. 命名规范

- **字段名**: 使用下划线分隔，例如 `temperature_1`, `pressure_2`
- **显示名称**: 使用中文描述，例如 "第一级温度", "出口压力"
- **参数类型**: 使用枚举值，例如 `TEMPERATURE`, `PRESSURE`

### 2. 单位转换

- **一致性**: 保持单位转换的一致性
- **精度**: 根据实际需要设置合适的精度
- **范围**: 设置合理的物理范围

### 3. 验证规则

- **完整性**: 包含所有必要的验证规则
- **错误信息**: 提供清晰的中文错误信息
- **性能**: 避免过于复杂的验证逻辑

### 4. 性能优化

```python
# 缓存映射结果
class CachedParameterMapper(ParameterMapper):
    def __init__(self):
        super().__init__()
        self._mapping_cache = {}
    
    def get_mapping(self, software_type: SoftwareType, param_type: ParameterType) -> Optional[ParameterMapping]:
        cache_key = (software_type, param_type)
        if cache_key not in self._mapping_cache:
            self._mapping_cache[cache_key] = super().get_mapping(software_type, param_type)
        return self._mapping_cache[cache_key]
```

## 故障排除

### 1. 常见问题

1. **映射未找到**: 检查软件类型和参数类型是否正确
2. **单位转换错误**: 验证单位转换表和公式
3. **验证失败**: 检查验证规则和输入值类型
4. **字段名不匹配**: 确认软件字段名拼写正确

### 2. 调试方法

