## 软件简介

UnveilChem是一款专为化工行业设计的AI文档智能解析与参数识别系统。该系统采用先进的人工智能技术，能够自动识别和提取化工文档中的工艺参数，并自动填充到目标应用软件中，实现从文档到软件的全流程自动化。

### 主要特色
- **智能识别**：基于深度学习的文档解析技术
- **多格式支持**：支持PDF、Word、Excel、图片等多种格式
- **高精度提取**：参数提取准确率达95%以上
- **一键对接**：与Aspen Plus等化工软件无缝集成
- **批量处理**：支持大批量文档并行处理

## 安装与配置

### 第一步：环境准备

1. **安装Python**
   - 访问Python官网（https://www.python.org/）
   - 下载Python 3.8+版本
   - 安装时勾选"Add Python to PATH"

2. **安装Node.js**
   - 访问Node.js官网（https://nodejs.org/）
   - 下载LTS版本
   - 按照安装向导完成安装

3. **验证安装**
   ```bash
   python --version
   node --version
   npm --version
   ```

### 第二步：软件安装

1. **下载软件包**
   - 从官方网站下载UnveilChem安装包
   - 解压到指定目录（如：C:\\UnveilChem）

2. **安装后端服务**
   ```bash
   cd backend
   pip install -r requirements.txt
   ```

3. **安装前端依赖**
   ```bash
   cd frontend
   npm install
   ```

### 第三步：系统配置

1. **配置文件设置**
   - 编辑`backend/app/config.py`文件
   - 设置数据库连接参数
   - 配置API密钥和访问权限

2. **初始化数据库**
   ```bash
   cd backend
   python -m app.database init
   ```

3. **启动服务**
   ```bash
   # 启动后端服务
   cd backend
   python -m app.main
   
   # 启动前端服务
   cd frontend
   npm run dev
   ```

## 快速入门

### 第一步：系统登录

1. **访问系统**
   - 打开浏览器，访问 http://localhost:3000
   - 如果是首次使用，需要先注册账号

2. **用户注册**
   - 点击"注册"按钮
   - 填写用户名、邮箱、密码等信息
   - 验证邮箱后完成注册

3. **用户登录**
   - 输入用户名和密码
   - 点击"登录"按钮进入系统

### 第二步：上传文档

1. **选择文档**
   - 点击"上传文档"按钮
   - 选择需要解析的文档（支持多选）
   - 支持的格式：PDF、DOC、DOCX、XLS、XLSX、JPG、PNG

2. **设置解析参数**
   - 选择文档类型（工艺文档、设备规格书、实验报告等）
   - 设置目标软件（Aspen Plus、DWSIM等）
   - 配置解析选项（OCR识别、表格提取等）

3. **开始解析**
   - 点击"开始解析"按钮
   - 系统显示解析进度
   - 解析完成后查看结果

### 第三步：参数提取与验证

1. **查看提取结果**
   - 在"解析结果"页面查看提取的参数
   - 参数按类别分组显示（温度、压力、流量等）
   - 显示每个参数的置信度

2. **参数验证**
   - 点击参数进行编辑和修正
   - 添加缺失的参数
   - 删除错误的参数

3. **单位转换**
   - 系统自动识别参数单位
   - 支持多种单位制转换
   - 可设置默认单位制

### 第四步：软件对接

1. **选择目标软件**
   - 在"软件对接"页面选择目标软件
   - 目前支持：Aspen Plus、DWSIM、ChemCAD

2. **参数映射**
   - 系统自动映射参数到目标软件字段
   - 可手动调整映射关系
   - 保存映射配置供后续使用

3. **自动填充**
   - 点击"自动填充"按钮
   - 系统自动打开目标软件
   - 完成参数自动填写

## 常见问题

### 安装问题

**Q: 安装过程中出现依赖包错误怎么办？**
A: 请确保网络连接正常，尝试使用国内镜像源安装依赖包：
```bash
pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
```

**Q: 系统启动后无法访问？**
A: 请检查防火墙设置，确保端口3000和8000已开放。同时检查服务是否正常运行。

### 使用问题

**Q: 文档上传失败是什么原因？**
A: 可能原因包括：文件格式不支持、文件大小超限、网络连接问题。请检查文件格式和大小，确保网络正常。

**Q: 参数提取准确率低怎么办？**
A: 可以尝试以下方法提高准确率：
1. 确保文档清晰度和格式规范
2. 选择正确的文档类型
3. 手动校正部分参数，系统会学习改进
4. 联系技术支持进行模型优化

**Q: 软件对接失败如何处理？**
A: 请检查：
1. 目标软件是否正确安装
2. 软件版本是否支持
3. 参数映射关系是否正确
4. 查看详细错误日志进行排查

### 性能问题

**Q: 处理大文件时系统卡顿？**
A: 建议：
1. 增加系统内存至16GB以上
2. 关闭其他占用资源的程序
3. 分批处理大文件
4. 使用更高性能的处理器

**Q: 如何提高批量处理效率？**
A: 可以：
1. 启用并行处理模式
2. 优化网络带宽
3. 使用SSD硬盘
4. 调整批处理大小
